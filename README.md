# deeplearning-review

深度学习期末复习指南——基于课程全部考点整理的交互式术语索引网站。

## 这是什么

将复习文档汇总成一个点开即用的复习网站，覆盖 **20 个大类、130+ 个核心术语**。

每个术语包含：
- **公式 / 流程**
- **通俗例子**
- **优缺点**
- **应用**
